# 简介

Cerise 是一个 AI 驱动的 Live2D 虚拟角色核心系统，提供多 AI Provider 支持、插件系统、情感引擎和记忆系统。

## 特性

- **多 AI Provider 支持** - 支持 OpenAI、Claude、Gemini 等多种 AI 服务商
- **情感引擎** - 基于规则和词典的情感分析系统
- **记忆系统** - 智能记忆管理，支持向量检索和知识图谱
- **插件系统** - 灵活的插件架构，支持 Python 和 Node.js
- **Live2D 集成** - 无缝集成 Live2D 模型
- **TTS 集成** - 多种 TTS 服务支持
