# Introduction

Cerise is an AI-powered Live2D virtual character core system, providing multi AI provider support, plugin system, emotion engine, and memory system.

## Features

- **Multi AI Provider Support** - Support for OpenAI, Claude, Gemini and more
- **Emotion Engine** - Rule and lexicon-based emotion analysis system
- **Memory System** - Intelligent memory management with vector retrieval and knowledge graph
- **Plugin System** - Flexible plugin architecture supporting Python and Node.js
- **Live2D Integration** - Seamless Live2D model integration
- **TTS Integration** - Multiple TTS service support
